# AI-Conversation-Tool
A speech-to-text pipeline powered by [Groq AI](http://groq.com/) that summarizes conversations, fact-checks in real time, and exposes the tool securely over HTTPS using Cloudflared.
### Features
```ini
[+] Converts Speech to Text
[+] AI Speech Summarizer
[+] AI Fat Checker
[+] Cloudflared Utilization ; Creates an https certificate
```
### Required Modules
```
express
axion
```
**⚠️ Cloudflared is not in this repository so you must install this manually online**
